## Carts API

> **Not implemented.** The Carts domain was not built as part of this evaluation. Cart/basket functionality is out of scope for the Sales API challenge. Sale creation (`POST /api/v1/sales`) accepts items directly in the request body, acting as the checkout step. There is no separate Carts service or controller in this codebase.
